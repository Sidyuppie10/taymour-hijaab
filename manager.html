📝 How to Manage Products (Secure Method):

To ADD a new product:
1. Open script.js in Notepad++ or VS Code
2. Find the products array at the top (line ~6)
3. Add a new line with your product:
{ id: 17, name: "New Product", price: 50000, image: "new-image.jpg", description: "Product description" },
4. Save the file and upload to your server
5. Refresh the website

To EDIT a product:
- Change the name, price, or description in script.js

To DELETE a product:
- Remove the product line from script.js

To ADD images:
- Upload image files to the images/ folder
- Use the exact filename in the product data
